How it works

Four steps, zero friction

From natural language to executed tool call in milliseconds.

01

You describe

Write a plain-English task like "list files in /tmp"

02

Claude analyzes

The router sends your request + full tool catalog to Claude for intelligent matching.

03

Tool executes

The matched tool (filesystem__list_directory) is called with the right parameters.

04

Result returns

You get a clean result back. For multi-step tasks, Claude chains calls automatically.

Get started

Up and running in 3 steps

Works with Windsurf, Cursor, Claude Desktop, and any MCP-compatible editor.

01

Install globally

Or run without installing using npx @vijaynkw/mcp-router

npm install -g @vijaynkw/mcp-router
02

Create config file

Create mcp-router.config.json in your project or home directory:

{
  "mcpServers": {
    "filesystem": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/your/path"]
    },
    "github": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": { "GITHUB_TOKEN": "ghp_your_token" }
    }
  }
}
03

Set API key & add to your editor

Set your Anthropic API key, then add mcp-router to your editor config:

export ANTHROPIC_API_KEY=sk-ant-... 
// Editor MCP config (Windsurf, Cursor, Claude Desktop)
{
  "mcpServers": {
    "router": {
      "command": "npx",
      "args": ["@vijaynkw/mcp-router"],
      "env": {
        "ANTHROPIC_API_KEY": "sk-ant-...",
        "MCP_ROUTER_CONFIG": "/path/to/mcp-router.config.json"
      }
    }
  }
}

Restart your editor — you'll see a single route tool instead of all individual tools.
